The Google Sheets integration allows you to create a Sheet within the connected Drive account.

## Parameters

1. **Parent Folder:** choose a folder to create the spreadsheet in
2. **Title:** the title of the created spreadsheet
3. **JSON Data (Array of Objects):** the content to populate the rows and columns, supports JSON

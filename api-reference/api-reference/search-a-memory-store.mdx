---
title: "Search a memory store"
description: "Perform a semantic search on a memory store. Returns the most relevant documents based on your query, with optional filtering by metadata."
api: GET https://api.airops.com/public_api/vector_stores/{memory_store_id}/search
contentType: application/json
---

### Authorizations

<ParamField header="Authorization" type="string" required>
API requests are authenticated using Bearer tokens. Include your API key in the Authorization header:

`Authorization: Bearer YOUR_API_KEY`

You can find your API key in your workspace settings.
</ParamField>



### Path parameters
<ParamField path="memory_store_id" type="string" required>
The Memory Store id

Example: `123`
</ParamField>

### Query parameters
<ParamField query="query" type="string" required>
The query to search for
</ParamField>

<ParamField query="topk" type="integer" min="1" max="1000">
The number of results to return

Example: `10`
</ParamField>
<ParamField query="filter" type="object" optional>
The filter to apply to the search. See /basic-setup/memory-stores/memory-stores-metadata#how-to-filter-on-metadata for more information.
<Expandable title="properties">
<ParamField query="key" type="string" optional/>
<ParamField query="value" type="string" optional/>
</Expandable>
</ParamField>

</ResponseField>

---
title: "Retrieve all executions for an Airops app"
description: "Returns all the executions for a given Airops app. You can use this endpoint to check the status/input/output of all the executions."
api: GET https://api.airops.com/public_api/airops_apps/:airops_app_id/executions
contentType: application/json
---

### Authorizations
<ParamField header="Authorization" type="string" required>
API requests are authenticated using Bearer tokens. Include your API key in the Authorization header:

`Authorization: Bearer YOUR_API_KEY`

You can find your API key in your workspace settings.
</ParamField>

### Query parameters
<ParamField query="airops_app_id" type="integer" required>
The Airops app ID to fetch executions from
</ParamField>

<ParamField query="cursor" type="string">
The cursor value to use for pagination
</ParamField>

<ParamField query="items" type="integer" min="1" max="100">
The amount of items to fetch

Example: `10`
</ParamField>

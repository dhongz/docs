---
title: "Download CSV / Check job status"
description: "Check the status of a CSV generation job and download the CSV file once it's ready. Returns the job status and download URL when complete."
api: GET https://api.airops.com/public_api/grids/{grid_id}/sheets/{sheet_id}/download_csv/{job_id}
contentType: application/json
---
### Authorizations

<ParamField header="Authorization" type="string" required>
API requests are authenticated using Bearer tokens. Include your API key in the Authorization header:

`Authorization: Bearer YOUR_API_KEY`

You can find your API key in your workspace settings.
</ParamField>


### Path parameters
<ParamField path="grid_id" type="string" required>
The Grid ID

Example: `789`
</ParamField>

<ParamField path="sheet_id" type="string" required>
The Sheet ID

Example: `101`
</ParamField>

<ParamField path="job_id" type="string" required>
The Job ID

Example: `230`
</ParamField>

<ResponseField name="Response" type="object">
